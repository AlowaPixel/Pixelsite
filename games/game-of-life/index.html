Info

For if you've never heard of the Game of Life, it is a cellular automaton created by mathematician John Conway in 1970.

Rules

Any given cell can have 1 of 2 states: alive or dead. Every so often we advance a generation and apply the following rules:
- A live cell with 2 or 3 live neighbors survives
- A dead cell with 3 live neighbors becomes alive
- Any other cell dies/stays dead
