Langton's ant

The simulation starts when you place the ant. You can wake cells or even move the ant while the simulation is running.
The ant will wrap around the screen if it gets to the edge.

Rules

At the start of the game, we name one specific cell the ant and when we advance a generation, we apply the following rules:
- If the current cell is alive, we kill it and turn 90° clockwise.
- If the current cell is dead, we wake it and turn 90° counter-clockwise.
- Then, we go one cell forward.

Controls

Left mouse button: Change state of cell
Right mouse button: Place ant and start
Esc: Remove ant
